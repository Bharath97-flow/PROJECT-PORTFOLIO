									    ADDRESS BOOK PROJECT




1.THE PROGRAM DOES NOT CONTAIN ANY STANDARD LIBRARY FUNCTIONS EXCEPT FOR THOSE OF IN stdio.h




2.USAGE OF ALL THE FUNCTIONS FROM string.h IS MIMICKED USING THE SELF CREATED FUNCTIONS THAT ACHIEVE SAME FUNCTIONALIY
        

		
		EX :
			strcmp()---------------------------------------->my_strcmp()
			
			strcpy()---------------------------------------->my_strcpy()

			strlen()---------------------------------------->my_strlen()




3.PROGRAM IS ROBUSTLY DESIGNED SO THAT IT CAN BE EXITED TO MAIN MENU AT ANY POINT IN TIME FROM ANY FUNCTION JUST BY ENTERING 0.
